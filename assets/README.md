# Assets Folder

Place your portfolio assets in this folder:

## Required Files:

1. **profile.jpg** - Your professional profile picture
   - Recommended size: 500x500px (square)
   - Format: JPG or PNG
   - Professional photo in good lighting

2. **resume.pdf** - Your current resume/CV
   - Keep it updated
   - 1-2 pages max for students
   - Professional formatting

3. **Project Screenshots:**
   - project1.jpg
   - project2.jpg
   - project3.jpg
   - project4.jpg
   - Recommended size: 1200x600px
   - Show your project interface/results

## Tips:

- Use compressed images to improve loading speed
- Maintain consistent aspect ratios for project images
- Use professional, high-quality photos
- Update regularly as you complete new projects

## Tools for Image Optimization:

- [TinyPNG](https://tinypng.com/) - Compress images
- [Squoosh](https://squoosh.app/) - Image optimization
- [Canva](https://www.canva.com/) - Create project thumbnails
- [Remove.bg](https://www.remove.bg/) - Remove backgrounds from profile photo

## Placeholder Images:

Until you have your own images, you can use placeholders from:
- [Unsplash](https://unsplash.com/)
- [Pexels](https://www.pexels.com/)
- [Placeholder.com](https://placeholder.com/)
